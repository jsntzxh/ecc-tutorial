什么是 ECC？

一套完整的 Claude Code 配置集合

  • 由 Anthropic 黑客松获奖者 Affaan Mustafa 开发
  • 经过 10+ 个月高强度日常使用打磨
  • 覆盖 14 种编程语言
  • 支持 7 个 AI 编程平台

Claude Code → AI 辅助开发平台

五大核心组件

组件数量功能
Rules（规则）89定义编码标准和规范
Skills（技能）147封装专业领域工作流

Commands（命令）68快捷操作入口
Hooks（钩子）—事件驱动的自动化

Rules：规则系统

三层架构

  • 通用规则 common/ — 10 个文件
  • 语言规则 <lang>/ — 14 种语言
  • 中文规则 zh/ — 中文翻译版

支持的语言

C C++ C# Dart Go Java Kotlin Perl PHP Python Rust Swift TypeScript Web

安装 ECC

方式一：npm 安装

npm install -g ecc-universal
npx ecc-install --profile full

方式二：Claude Code 插件

/plugin install everything-claude-code@everything-claude-code
